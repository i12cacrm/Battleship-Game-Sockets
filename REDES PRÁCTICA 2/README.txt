Compilacion:
	-Forma 1: make
	-Forma 2 (manual):
	 	gcc servidor.c funciones.c -o servidor
	 	gcc cliente.c funciones.c -o cliente
	 	
	 	
	 	
	 	
Ejecucion:
	SERVIDOR->./servidor
	CLIENTE->./cliente

2. Los datos de los registro se guardan en "usuarios.txt"
3.. El formato de "usuarios.txt" es el siguiente:
							usuario1 contrasena1
							usuario2 contrasena2
							...
